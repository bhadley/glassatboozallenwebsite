Here is a simple data entry tool where you can enter a linear sequence of information that you'd like displayed on Glass.

Glass will show the first step of your instructions as soon as you look at a visual cue, which can either be a QR code associated with your instructions or an image. Select which visual cue to use. Once you submit, you can download and save the image.

Here you will enter the sequential information. Each step should start with the step number, a period, and the step then a new line and the next step.

If you'd like Glass to set a timer on a particular step, use this format on that step: #. step text {timer: description of timer, number of seconds}
